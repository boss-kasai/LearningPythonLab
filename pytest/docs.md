# Pytestの勉強

## テスト駆動開発の流れ

1. レッド：テストコードを書いて失敗
2. グリーン：とりあえず、テストをクリアするようする(クソコードOK)
3. リファクタリング：テストをクリアする状態で、コードを綺麗にリファクタリングする

## このファイルの流れの説明

```sre/converter_1.py``` -> シンプルなpytestの書き方
```sre/converter_2.py``` -> 複数条件
```sre/converter_3.py``` -> 2つの命名規則に対応
```sre/converter_4.py``` -> 複数関数や複雑な条件
